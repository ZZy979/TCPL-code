# 运行方式
方式一：复制到Visual Studio运行

方式二：在命令行中使用gcc编译，例如：
```bash
# Windows
> mkdir output
> gcc -o output\hello_world.exe ch1\hello_world.c
> output\hello_world.exe

# Linux
$ mkdir output
$ gcc -o output/hello_world ch1/hello_world.c
$ output/hello_world
```
